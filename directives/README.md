# Directives

Use directives sparingly.

## Default Read Order

1. `AGENTS.md` for repo-wide operating rules
2. `master_directive.md` for the execution architecture
3. One task-specific directive only if the task actually needs it

## Important

- `OUTLINE.md` is the main router for the repo
- `.agent/skills/` contains the smaller module guides
- `_archive/` contains retired or historical docs

Do not load every directive by default.
