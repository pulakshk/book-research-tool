# Local Skills

Use the smallest skill that matches the task so the model loads less context.

## Available Skills

- `book_research_pipeline/` - Router skill for the whole workspace
- `research_and_scraping/` - Discovery, crawl, and raw metadata collection
- `validation_and_analysis/` - Enrichment, audits, and scoring
- `outreach_operations/` - Contact discovery, outreach prep, and licensing sheets

## Recommended Flow

1. Read `OUTLINE.md`
2. Pick one module skill
3. Fall back to `book_research_pipeline/` only when the task spans multiple modules
