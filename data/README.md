# Data Directory

Canonical shared datasets live here.

## Highest-Signal Files

- `unified_book_data_enriched_ultra.csv` - Main ultra-enriched master
- `unified_book_data_enriched_final.csv` - Finalized shared dataset
- `author_contacts_all_subgenres.csv` - Combined author contact master
- `ice_hockey_full_merged.csv` - Ice hockey master dataset

## Subfolders

- `subgenre_masters/` - One master CSV per subgenre
- `subgenre_outreach/` - Outreach-ready CSVs per subgenre
- `genre_expansion/` - Genre expansion master outputs
- `master_backups/` - Workbook backups

## Rule Of Thumb

If you need the latest shared data, start here before looking inside `_archive/` or older report folders.
