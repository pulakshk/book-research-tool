# Genre Crawl

This folder contains tightly coupled crawl-stage scripts.

## Use It For

- Single-genre crawl runs
- Scout discovery and aggregation
- Genre-specific repair steps

## Important

- Many scripts import neighboring files directly.
- Treat this as a mini-pipeline, not a bag of independent utilities.
- Historical narrative summaries were moved to `_archive/reports/subgenre-pipeline/genre-crawl/`.
